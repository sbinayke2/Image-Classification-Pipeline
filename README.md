# Image Classification Pipeline with MLflow

## Overview
This project is an end-to-end image classification system using TensorFlow/Keras and EfficientNet with MLflow for experiment tracking and FastAPI for deployment.

## Features
- Image Classification
- Transfer Learning using EfficientNet
- MLflow Experiment Tracking
- FastAPI Deployment
- Model Evaluation

## Tech Stack
- Python
- TensorFlow/Keras
- EfficientNet
- MLflow
- FastAPI
